# Supply Chain Control Tower

This component provides a real-time monitoring dashboard for the **Agentic AI Supply Chain** system.

## Key Features
- **Real-time Inventory Monitor**: View current stock, predicted demand, and agent-driven reorder decisions.
- **System Simulation**: Input real-time data to test the Agent Engine.
- **System Logs**: View real-time feedback from Azure Function endpoints.

## Tech Stack
- **HTML5/CSS3**: Modern, responsive UI with glassmorphism aesthetics.
- **Vanilla JavaScript**: Lightweight client-side logic for API interaction.
- **Google Fonts**: Inter font family for a premium look.

## Usage
Open `frontend/index.html` in a web browser to access the dashboard. Ensure the backend APIs (Supply Chain API and Disaster Response Service) are running to see live data.
